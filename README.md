# GPU-NET

A full-stack application with Ethereum smart contract backend deployment.

---

## Getting Started

### Clone the Repository
```bash
git clone https://github.com/chakri-chax/GPU-NET
cd GPU-NET
npm install
npm run dev
```

## Backend Deployment

```bash
cd backend
npm install
npx hardhat run scripts/deploy_fullsuite.js --network <network>
```
